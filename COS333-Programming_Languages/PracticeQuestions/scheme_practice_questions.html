Q1 keepNonNegatives
5 pts · Filter ▼

Write a Scheme function named keepNonNegatives, which receives one parameter: a simple list of numeric atoms. The function should yield (not print) a list containing only the non-negative values from the parameter list, in their original order.

Examples

(keepNonNegatives '())
; => ()

(keepNonNegatives '(-3 -1 -5))
; => ()

(keepNonNegatives '(-4 0 7 -2 3))
; => (0 7 3)
Your Answer Scheme
Hint: This is Pattern A (Filter). Keep if (>= (car lst) 0), skip otherwise. Base case returns '().
Memo Solution Filter A — keep non-negatives 
(define (keepNonNegatives lst)
  (cond
    ((null? lst) '())
    ((>= (car lst) 0)
     (cons (car lst) (keepNonNegatives (cdr lst))))
    (else
     (keepNonNegatives (cdr lst)))
  )
)
Trace for (-4 0 7)

-4 < 0 → skip · 0 ≥ 0 → cons · 7 ≥ 0 → cons → (0 7)

Your Answer
Memo
(define (keepNonNegatives lst) (cond ((null? lst) '()) ((>= (car lst) 0) (cons (car lst) (keepNonNegatives (cdr lst)))) (else (keepNonNegatives (cdr lst))) ) )
