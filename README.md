## Structure

```
index.html              ← Home page (all lectures + guides overview)
style.css               ← Shared stylesheet
notes/
  index.html            ← Lecture notes index
  L01.html … L21.html   ← All 21 lecture pages
guides/
  index.html            ← Guides index
  01_opengl.html        ← OpenGL State Machine guide
  02_shaders.html       ← Vertex & Fragment Shaders guide
  03_questions.html     ← Question Bank (10 MCQ + 3 long Qs per lecture)
  04_situations.html    ← Situation-Based Guide
  05_math.html          ← Math Guide
```


## Content

Based on **Shirley & Marschner, *Fundamentals of Computer Graphics*** — Chapters 1, 3, 4, 5, 7, 8, 9, 11, 12, 14, 16, 17, 19, 22, 23

Covers all 21 lectures across 4 parts:
- **Part 1:** Foundations & Rendering (L01–L05)
- **Part 2:** Hardware, Pipeline & Transforms (L06–L10)  
- **Part 3:** Viewing, Assets & Textures (L11–L16)
- **Part 4:** Advanced Topics (L17–L21)
