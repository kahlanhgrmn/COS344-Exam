Concepts of Programming Languages

Quick Pattern Reference

Revision
A — Filter

Keep or remove elements. Output ≤ input length. Base: '() / [].

"keep only X" · "remove all Y"

B — Map

Transform every element. Output length = input. Base: '() / [].

"double all" · "square each"

C — Filter+Map

3-way split. Exam favourite. Base: '() / [].

"X unchanged, Y transformed, Z omitted"

D — Accumulate

Return a number (count or sum). Base: 0.

"return the count of" · "return the sum of"
